    Control Flow Statements in Java¶

    What are Control Flow Statements?¶

    Control flow statements determine the order in which program instructions are executed.

    By default, Java executes statements sequentially (top to bottom).
    Control flow statements allow a program to:

    • make decisions
    • repeat actions
    • transfer execution control

    Control flow statements are classified into three categories:

    1. Decision-Making Statements
    2. Looping Statements
    3. Branching (Jump) Statements

    Decision-Making Statements¶

    Decision-making statements allow the program to choose which block of code should execute based on a condition.

    Java provides two decision-making statements:

    • if
    • switch

    if Statement¶

    The if statement executes a block of code only when the given condition evaluates to true.

    Simple if Statement¶

    Executes code only if the condition is true.

    Syntax¶

    if (condition) {
    // statements executed only if condition is TRUE
}

    Example¶

    int age = 20;

if (age >= 18) {
    System.out.println("Eligible to vote");
}

    if-else Statement¶

    Used to choose between two alternative blocks.

    • if block → executes when true
    • else block → executes when false

    Syntax¶

    if (condition) {
    // executes if TRUE
} else {
    // executes if FALSE
}

    Example¶

    int number = 5;

if (number % 2 == 0) {
    System.out.println("Even");
} else {
    System.out.println("Odd");
}

    else-if Ladder¶

    Used when multiple conditions must be checked.

    • Conditions are evaluated from top to bottom
    • The first true block executes
    • Remaining conditions are skipped

    Syntax¶

    if (condition_1) {
    // executes if condition_1 is TRUE
} else if (condition_2) {
    // executes if condition_2 is TRUE
} else if (condition_3) {
    // executes if condition_3 is TRUE
} else {
    // executes if all conditions are FALSE
}

    Example¶

    int marks = 85;

if (marks >= 90) {
    System.out.println("Grade A");
} else if (marks >= 75) {
    System.out.println("Grade B");
} else if (marks >= 50) {
    System.out.println("Grade C");
} else {
    System.out.println("Fail");
}

    switch Statement¶

    The switch statement compares a variable against multiple fixed constant values.

    It is often cleaner and more readable than multiple else-if statements.

    Key Points¶

    • Works with: byte, short, int, char, String, enum
    • Uses case labels
    • break prevents fall-through
    • default runs when no case matches

    Syntax¶

    switch (key) {

    case value_1:
        statement_1;
        break;

    case value_2:
        statement_2;
        break;

    default:
        default_statement;
}

    Example¶

    int day = 2;

switch (day) {
    case 1:
        System.out.println("Monday");
        break;

    case 2:
        System.out.println("Tuesday");
        break;

    default:
        System.out.println("Invalid day");
}

    Summary¶

    Statement Purpose
    if Single condition check
    if-else Two-way decision
    else-if ladder Multiple conditions
    switch Multiple fixed values (cleaner alternative)
